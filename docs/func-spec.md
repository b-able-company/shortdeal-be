## 1. 인증 및 계정 관리

### 1.1 회원가입

| 항목 | 내용 |
| --- | --- |
| 화면 | `/join` |
| 접근 권한 | 비로그인 |

**기능 상세**

| ID | 기능명 | 설명 |
| --- | --- | --- |
| AUTH-001 | 역할 선택 | 제작사/바이어 중 선택 (필수, 가입 후 변경 불가) |
| AUTH-002 | 기본 정보 입력 | 이메일, 비밀번호, 이름 입력 |
| AUTH-003 | 이메일 중복 검증 | 실시간 중복 확인 |
| AUTH-004 | 비밀번호 유효성 검증 | 최소 8자, 영문+숫자 조합 |
| AUTH-005 | 이용약관 동의 | 필수 약관 체크박스 |

**입력 필드**

| 필드명 | 타입 | 필수 | 유효성 |
| --- | --- | --- | --- |
| 역할 | Radio | ✅ | producer / buyer |
| 이메일 | Email | ✅ | 이메일 형식, 중복 불가 |
| 비밀번호 | Password | ✅ | 8자 이상, 영문+숫자 |
| 비밀번호 확인 | Password | ✅ | 비밀번호와 일치 |
| 이름 | Text | ✅ | 2-50자 |

**비즈니스 규칙**

- 가입 완료 시 역할에 따라 온보딩 페이지로 리다이렉트
- 제작사 가입 시 부스(Booth) 자동 생성

**기능 상세**

| ID | 기능명 | 설명 |
| --- | --- | --- |

| BRW-002 | 키워드 검색 | 제목, 제작사명, 시놉시스 통합 검색 |
| BRW-003 | 장르 필터 | 다중 선택 가능 |

| 제작사명 | 클릭 시 부스 이동 |

| 파라미터 | 타입 | 설명 |
| --- | --- | --- |
| `q` | string | 검색 키워드 |
| `genre` | string[] | 장르 필터 |
| `price_min` | number | 최소 희망가 |
| `price_max` | number | 최대 희망가 |
| `page` | number | 페이지 번호 |

## 4. 오퍼 관리

### 4.1 오퍼 작성 (바이어)

| 항목 | 내용 |
| --- | --- |
| 화면 | `/content/:contentId/offer` 또는 모달 |
| 접근 권한 | 바이어 |

**기능 상세**

| ID | 기능명 | 설명 |
| --- | --- | --- |
| OFR-001 | 제안가 입력 | 필수, 숫자 |
| OFR-002 | 메모 입력 | 선택, 최대 500자 |
| OFR-003 | 유효기간 선택 | 필수, 7/14/30일 선택 |
| OFR-004 | 콘텐츠 정보 확인 | 제목, 희망가 표시 (읽기 전용) |
| OFR-005 | 오퍼 제출 | 확인 후 전송 |
| OFR-006 | 중복 오퍼 방지 | 동일 바이어가 동일 콘텐츠에 pending 오퍼가 있으면 제출 불가 (다른 바이어는 별도 제출 가능) |

**입력 필드**

| 필드명 | 타입 | 필수 | 유효성 |
| --- | --- | --- | --- |
| 제안가 | Number | ✅ | >0 |
| 메모 | Textarea | ❌ | ≤500자 |
| 유효기간 | Select | ✅ | 7/14/30일 |

**오퍼 상태**

- `pending`: 대기 중
- `accepted`: 수락됨
- `rejected`: 거절됨
- `expired`: 만료됨
